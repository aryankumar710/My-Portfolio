Food Website

Project Type: Practise
Tools: HTML, CSS, Bootstrap

Developed a fully responsive single-page food website using Bootstrap, focusing on seamless user experience, clean layout, and modern design principles.

Github Link

About Project

This project gave me the opportunity to apply my Bootstrap skills, which I had recently picked up through tutorials. I explored various Bootstrap utilities like d-flex, alignment classes, and also experimented with layout techniques to improve responsiveness and structure. I also learned about implementing a parallax effect, which added subtle visual depth to certain sections and enhanced the overall feel of the site. To make the typography more responsive, I used rem units instead of px for font sizing, which helped the text scale better across different screen sizes. Overall, this project helped me understand how to build more adaptive and visually dynamic websites using Bootstrap efficiently.
